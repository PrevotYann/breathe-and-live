# Checklist de couverture des règles (version physique -> addon Foundry)

Ce document sert de vérification rapide : les règles de la version physique sont présentes dans l'addon via la transcription intégrale ci-dessous.

- Source incluse : `resources/rules/regles-physiques-v1.1.txt`
- Origine : export texte du Google Slides partagé.

## Domaines explicitement couverts

- Souffles (standards + homebrew) : sections *Techniques de Souffle* et sous-sections dédiées.
- Types de personnages : pourfendeurs, démonistes, démons jouables, PNJ, sous-classes.
- Combat : économie d'action, effets de combat, attaques de base, mutilation, conditions.
- NPC/PNJ : règles et intégration des profils non-joueurs présentes dans la table des matières/sections dédiées.
- Expérience et progression : montée de niveau, rangs, évolution de ressources.
- Armes et équipements : outils/objets, armes, vêtements, nourriture, médical.
- Moyens de locomotion : section *Transports*.
- Classes : classes principales + sous-classes.
- Points de réaction : définition, calcul, régénération, usages.
- Mort : section dédiée *Mort*.
- États avancés : TCB constant, Forme marquée, Monde transparent, Lame rouge.

## Note d'usage

Le fichier source inclut tout le contenu textuel exportable de la présentation; il peut être utilisé comme référence de règles en jeu (copie vers Journal Foundry, handouts, ou compendium interne).
